# Node api seed

## Getting Started

Install mongodb 

`brew update`
`brew install mongodb`

Install the module dependencies with: 

`npm install`
`sudo npm install -g mocha`

Configure project for developing:

`grunt configure`

## Running server and mongod

Run  `grunt`

## Testing

Run server and then run `grunt test`
